# 项目总览与快速启动

> 说明：此文件为基于本次会话已读取内容的恢复版，优先恢复关键结构、启动步骤与文档索引。

一个面向论文展示和原型验证的 DevSecOps 审计平台，当前已具备可运行的前后端系统，而不是停留在概念或占位实现。

## 当前能力

- 用户注册、登录、JWT 鉴权、当前用户信息获取
- SAST：上传 Python 文件/压缩包，调用 Bandit 执行真实静态审计
- DAST：对本地或授权白名单目标调用 Wapiti 执行真实动态扫描
- Celery + Redis 异步任务执行
- 任务列表、分页筛选、详情查看、取消、重跑
- 审计日志查询（管理员）
- 针对重点问题的 AI 解读缓存
- 扫描结果入库、摘要统计、OWASP 映射
- SAST / DAST 报告下载与 HTML / PDF 导出

## 技术栈

- 后端：Flask、SQLAlchemy、Flask-JWT-Extended、Celery
- 前端：Vite、Vue 3、TypeScript
- 扫描器：Bandit、Wapiti
- 基础设施：MySQL、Redis、Docker Compose

## 快速启动

### 1. 推荐：一键启动联调环境

```powershell
cd D:\ziliao\dev
powershell -ExecutionPolicy Bypass -File .\start_demo_env.ps1
```

默认地址：

- 前端：`http://127.0.0.1:5173`
- 后端健康检查：`http://127.0.0.1:5000/api/health`
- DAST 靶场：优先 `http://127.0.0.1:3000`，若 `3000` 被占用则脚本会自动回退到 `http://127.0.0.1:3100`

附加说明：

- 运行状态文件：`tmp/run-control/demo-runtime.json`
- 启动日志目录：`tmp/run-control/logs`
- 如果本机相关端口上已经有可用服务，脚本会优先复用，而不是强行重启

### 2. 如需手动启动，先启动 Redis 和 Juice Shop

```powershell
cd D:\ziliao\dev
& 'C:\Program Files\Docker\Docker\resources\bin\docker.exe' compose up -d redis juice-shop
```

默认 DAST 演示靶场：`http://127.0.0.1:3000`

若 `3000` 已被占用，也可改用：`http://127.0.0.1:3100`

### 3. 安装后端依赖

```powershell
cd D:\ziliao\dev\backend
D:\anaconda\python.exe -m pip install -r requirements.txt
```

### 4. 初始化 / 升级数据库

```powershell
cd D:\ziliao\dev\backend
D:\anaconda\python.exe -m flask db upgrade
```

## 一周增强入口补充

- SCA 依赖漏洞扫描：在“新建扫描”页选择 `SCA`，上传 `requirements.txt` 或包含该文件的 ZIP，可查看依赖漏洞、可修复版本并导出报告。
- DAST 授权边界：创建 DAST 前必须勾选授权确认，后端会记录目标主机、解析 IP、命中策略和授权状态。
- CI/CD 脚本：`scripts\ci_security_scan.py` 可被本地脚本或示例流水线调用，用退出码表达质量门禁结果。
- 队列统计接口：`/api/metrics/task-summary` 用于工作台展示真实任务状态分布。
- 管理员看板：管理员登录后可访问 `/admin`，查看全局任务、风险、审计日志和 DAST 授权目标。
